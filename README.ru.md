## JustFCbot  
---

Бот созданный по-приколу для реализации функций которых мне не хватает в самом Telegram и которые возможно реализовать с помощью BotAPI от Telegram.
Функции реализованные на данный момент:
- Сообщения user-to-user в любых чатах где доступен inline-мод у бота(боту необязательно быть в группе)
- Создание событий (событие - время создания) (просто интересно было это реализовать)
